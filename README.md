# IP知识库（IP基础概念 + 网络原理 + 代理IP解析）

系统化整理 IP 地址与网络基础知识的学习仓库，涵盖 IP 地址查询、公网IP与内网IP区别、IPv4 / IPv6、静态IP与动态IP、IP变化机制、IP归属地原理，以及代理IP（SOCKS5 / HTTP / L2TP）基础知识与应用场景。

本项目适用于网络基础学习、跨境电商、代理IP使用者以及网络工具开发与测试人员。

---

## 📌 为什么要了解IP？

在真实网络环境中，IP决定了你的：

- 🌍 网络身份与地理位置
- 🔐 访问权限与风控识别
- 📊 账号安全与行为关联判断
- ⚙️ 跨境业务网络稳定性
