# IP知识库 | IP基础知识体系（IP地址 / IPv4 / IPv6 / 公网IP / 内网IP / 代理IP完整指南）

IP知识库是一个系统化整理互联网IP基础知识与网络原理的专业学习项目，涵盖IP地址、IPv4/IPv6、公网IP与内网IP、静态IP与动态IP、住宅IP、代理IP（SOCKS5/HTTP）等核心内容。

本项目适用于网络基础学习、跨境电商网络环境搭建、代理IP使用认知、IP检测与网络风控分析、多账号运营环境搭建等场景。

# 九、核心总结

IP是互联网通信的基础标识体系，其变化机制、类型分类以及代理技术共同构成现代网络环境的核心基础。

理解IP，本质是理解：

网络如何识别你、如何区分你、如何限制你，以及如何改变你。
